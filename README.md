# Alura_lingua

# 🌍 Language School - Landing Page

Esta é uma landing page moderna e responsiva desenvolvida para uma escola de idiomas fictícia. O projeto foca em uma interface limpa, intuitiva e otimizada para conversão de novos alunos.



## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes tecnologias:

*   **HTML5:** Estruturação semântica do conteúdo.
*   **CSS3:** Estilização, layout (Flexbox/Grid) e animações.
*   **Google Fonts:** Tipografia personalizada para melhor legibilidade.


## 📋 Funcionalidades

*   **Design Responsivo:** Adaptado para dispositivos móveis, tablets e desktops.
*   **Lista de Cursos:** Cards detalhando os idiomas oferecidos.
*   **Formulário de Contato:** Captura de leads para matrículas.
*   **Efeito Hover:** Feedback visual em botões e links de navegação.
